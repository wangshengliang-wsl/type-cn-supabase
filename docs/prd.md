### 《English‑to‑Chinese Typing Course》MVP 需求文档


---

## 1. 产品定位

一款面向海外中文学习者的 **分级中文打字学习平台**。

* **核心价值**：用“打字测验 + 即时纠错 + 游戏化激励”的方式，帮助零基础或进阶学习者快速掌握拼音输入与高频词/句。
* **商业模式**：内容付费 + 订阅会员。

> **MVP 范围**：首发仅支持 Web 端、英文界面；功能压缩至最小可用形态。

---

## 2. 目标用户

| 类型     | 典型画像              | 需求痛点                         |
| ------ | ----------------- | ---------------------------- |
| 初级学习者  | 英语母语者，刚开始接触汉语     | 想先学会基础打招呼、常用动词等，需要循序渐进，有明确反馈 |
| 考试/留学党 | 赴华交换或准备 HSK 考试的学生 | 需要大量词汇记忆与自测                  |
| 兴趣向    | 喜爱中华文化、想看原版弹幕/社媒  | 需要有趣、短平快的练习场景                |

---

## 3. 用户旅程（MVP）

1. 访客 → 落地页 → 注册
2. 免费用户 → 主页 → 体验免费课程 → 付费提示
3. 付费会员 → 解锁全部课程 → 连续学习 → 完成课程 → 鼓励弹框 → 返回课程列表

---

## 4. 信息架构

```
- Landing Page（公开）
- Auth（仅邮箱 + 密码注册／登录）
- App Shell（登录后）
  - 侧边栏
    · 主页 Home
    · 课程包商城 Course Store
    · 升级会员 Membership
  - 顶部栏
    · 用户头像 / 退出
- 课程详情页（弹窗或独立路由）
- 学习界面 Lesson Player
- 完成弹框 Congrat Dialog
```

### 5.2 Auth

* **注册 / 登录**：仅支持 **邮箱 + 密码**。
* 表单实时校验；成功后重定向至注册前意图页。
* 忘记密码：邮件重置。

| 方案   | 价格     | 权益         |

> MVP 只需显示定价与支付按钮，后端结算可采用占位逻辑。

### 5.5 Course Store – 课程卡结构 & 多选标签筛选

课程商城页面字段与交互：

| 元素       | 字段映射             | 说明                                          |
| -------- | ---------------- | ------------------------------------------- |
| 封面图      | `cover`          | 4∶3 自适应裁切                                   |
| **分类标签** | `tag`            | 单值分类值：*Greeting* / *DailyLife* / *Dining* … |
| 价格角标     | 鉴权结果             | `FREE`（未付费可学） / `PRO`（需订阅） / `OWNED`（已购买）   |
| 标题       | `title_en`       | 主标题                                         |
| 子标题      | `title_zh` (可选)  | 中文辅助标题                                      |
| 描述摘要     | `description_en` | 截取首 45 字符，Hover/点击可查看完整描述                   |
| 学习进度     | 用户进度             | 已完成题数 / 总题数，登录后显示                           |

**筛选交互**

* **默认视图**：加载时显示全部课程卡。
* 页面顶部放置 **Tag Selector**（Chip / Checkbox 形式，可多选）。

  * 单击 Chip → 进入选中态（高亮）。
  * 可连续勾选多枚 Chips；前端使用 **并集过滤**：展示 `tag` 属于「当前选中集合」的所有课程。
  * 再次点击已选 Chip 可取消；当无任何 Tag 被选中时，自动恢复「全部」视图。
* 过滤逻辑完全在前端完成，无需额外接口调用；切换过渡使用淡入淡出动画提升体验。

### 5.6 Lesson Player（学习界面）（学习界面）

新增显示字段，保持原有交互：

1. **顶部栏**

   * 分类 Chip：显示 `tag`（如 *Greeting*）
   * 课程标题：`title_en`
   * 进度条 + 退出按钮

2. **课程简介（可折叠）**

   * `description_en`（帮助学习者了解学习目标）

3. **题目区域**（逻辑不变）

   * 默认英文提示 `en`
   * 输入框（拼音）
   * 正确后显示 `zh` + `py`
   * 自动双发音 & Enter 校验流程

其他交互、快捷键与完成弹框保持上一版描述。

（学习界面）

| 区域    | 元素                             | MVP 交互                                                                                                                                                               |
| ----- | ------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 顶部    | 进度条 + 退出                       | ESC 触发退出确认                                                                                                                                                           |

题目数据示例：`{"en":"I", "zh":"我", "py":"wo3"}`
可接受写法：`nihao` / `ni hao` / `ni3hao3` / `ni3 hao3`（大小写 & 空格不敏感）。

## 6. 业务规则

| 主题   | 规则                     |
| ---- | ---------------------- |
| 课程结构 | 课程 → 多节课（Lesson）→ 10 题 |
| 题目答案 | 4 种默认可接受写法；错误提示显示正确拼音  |
| 计分逻辑 | 正确 +1，错误 0             |
| 授权策略 | 免费课程对所有用户开放；付费课程需订阅解锁  |
